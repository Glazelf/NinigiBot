# Security Policy

## Supported Versions

| Discord Version | Supported |
| ------- | ------------------ |
| Stable (Desktop and mobile) | :white_check_mark: |
| Canary (Desktop and mobile) | :white_check_mark: |
| Extensions (e.g. BetterDiscord) | :x: |
| Other (e.g. older versions) | :x: |
